# College Football Conference

This dataframes is used internally to identify conference names and use
the abbreviation in the function call for data

## Usage

``` r
cfb_conf_types_df
```

## Format

A data frame with 11 rows and 4 variables:

- id:

  Referencing conference id

- name:

  Conference name

- short_name:

  Short name for Conference

- abbreviation:

  Conference abbreviation

## Source

<https://api.collegefootballdata.com>
